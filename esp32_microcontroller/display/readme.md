#   liquid crystal display

-   comes with a parallel and also I2C connection
    -   paralell:   needs up to all 16 pins
        -   usually, "only" 10 pins are minimal required
        -   doesn't comes with a potentiometer for contrast, unless this is already plugged in
        -   doesn't comes with background control for the display
    -   I2C: needs only 4 pins
        -   needs a specific address: **0x27**
-   comes with 16x2, 16x4, 20x2, ... aspect ratio
